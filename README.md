# Socket I/O and WebSockets Example
### Setup:

1- Install all dependencies:

```npm install```

2- Configuring ports:

``` \app\public\js\main.js```

``` \app\server\chichat.js```

``` port: '8013' ```

3- Launch server:

```cd .\app\server```

```npm start```

4- Open the ```index.html``` in your browser.

5- Start typing!

6- Open more browser instances to emulating a group chat room.
